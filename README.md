# 📊 Data Analyzer - 數據分析工具

強大的數據可視化與分析工具，支援 CSV 和 JSON 文件上傳。

## ✨ 功能特色

### 📈 圖表可視化
- **折線圖**：顯示趨勢變化
- **長條圖**：比較數值大小
- **圓餅圖**：展示比例分布
- 即時切換圖表類型
- 多指標同時顯示
- 響應式圖表大小

### 📊 統計分析
- 平均值（Average）
- 中位數（Median）
- 最小值/最大值
- 總和
- 趨勢分析（增長/下降）

### 📁 文件支援
- 上傳 CSV 文件
- 上傳 JSON 文件
- 自動解析數據
- 示例數據預覽

### 💾 報表匯出
- 匯出完整分析報表（JSON 格式）
- 包含原始數據和統計結果
- 時間戳記錄

### 📱 其他功能
- 原始數據表格顯示
- 指標選擇/過濾
- 響應式設計
- 優雅的 UI/UX

## 🚀 本地開發

```bash
npm install
npm run dev
```

## 📦 部署到 GitHub Pages

```bash
npm run build
```

## 🛠️ 技術棧

- React 18
- Tailwind CSS
- Recharts (圖表庫)
- PapaParse (CSV 解析)
- Vite

## 📖 使用說明

### 1. 上傳數據
- 點擊「選擇文件」上傳 CSV 或 JSON
- 或使用「使用示例數據」查看效果

### 2. 選擇圖表類型
- 折線圖：適合時間序列
- 長條圖：適合比較分析
- 圓餅圖：適合比例展示

### 3. 查看統計
- 自動計算所有數值欄位的統計數據
- 趨勢箭頭顯示增減

### 4. 匯出報表
- 點擊「匯出報表」下載完整分析

## 📝 數據格式示例

### CSV 格式
```csv
month,sales,expenses,profit
1月,4000,2400,1600
2月,3000,1398,1602
```

### JSON 格式
```json
[
  { "month": "1月", "sales": 4000, "expenses": 2400, "profit": 1600 },
  { "month": "2月", "sales": 3000, "expenses": 1398, "profit": 1602 }
]
```
